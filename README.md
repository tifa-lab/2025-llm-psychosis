# Evaluation of large language model chatbot responses to psychotic prompts: analytic code

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.17575242.svg)](https://doi.org/10.5281/zenodo.17575242)

To use:

1. Get [the numeric data from Dryad](https://doi.org/10.5061/dryad.x0k6djj00) and put it in `./data`
2. Pull our [container](https://quay.io/tifalab/2025-llm-psychosis).
3. Run the notebook to reproduce our analyses. Figures and tables will be written to `./output`
